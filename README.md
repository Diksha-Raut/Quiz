# Quiz Application

This is a simple quiz application built using HTML, CSS, and JavaScript. It allows users to take quizzes on various topics and see their scores at the end.

## Features

- Multiple-choice questions
- Score tracking
- User-friendly interface

## Technologies Used

- HTML
- CSS
- JavaScript
